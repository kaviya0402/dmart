# DMart Sales Dashboard - Power BI Project

## Project Objective
To analyze DMart sales performance and identify trends in revenue, profit, and customer behavior using Power BI.

## Tools Used
- Power BI
- MS Excel

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Category-wise Sales
- Region-wise Performance

## Dashboard Features
- Interactive charts and visuals
- Filters and slicers for dynamic analysis
- Category and region-level insights

## Configuration Details
- Data imported and cleaned in Power BI
- Relationships created between tables
- DAX measures used for KPI calculations
- Dashboard designed for business insights

## Conclusion
This dashboard helps understand DMart sales trends and supports data-driven decision making.
